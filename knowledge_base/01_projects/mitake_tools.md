# 專案：三竹股市開發流程自動化工具

## 背景問題
原有的三竹股市 PC 版開發流程中，"包版" (Build & Release) 過程耗時且容易出錯，且 Excel 設定檔轉入程式的過程依賴人工，溝通成本高。

## 解決方案
### 1. Python 自動化包版工具
* **功能:** 透過 Python 腳本串接編譯器與版控系統，一鍵完成編譯、打包與版號更新。
* **成效:** 將原本繁瑣的包版時間**降低了 90%**，讓團隊能專注於開發。

### 2. Excel 配置轉檔系統
* **功能:** 設計自動化腳本，讀取企劃維護的 Excel 規格書，直接生成程式所需的 `.ini` 設定檔。
* **成效:** * 消除人工複製貼上造成的錯誤。
    * 程式與企劃文件保持同步，大幅縮短團隊溝通時間。
