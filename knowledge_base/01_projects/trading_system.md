# 專案：自動化程式交易回測系統 (Automated Trading System)

## 專案概覽
* **時間:** 2022 - 2024
* **角色:** 獨立開發者 (Solo Developer)
* **簡介:** 一套自行研發的完整交易生態系，具備策略回測、歷史數據分析、自動下單以及即時訊息推播功能。

## 技術架構 (Tech Stack)
* **語言:** C#
* **框架:** PureMVC, WPF UI
* **資料庫:** (推測為 MySQL/PostgreSQL，依據技能列表)
* **API 串接:** 券商行情/下單 API, LINE Notify API, Slack API

## 功能細節
1.  **歷史回測:** 將交易邏輯程式化，驗證策略在歷史數據中的表現。
2.  **自動執行:** 串接券商 API 實現全自動無人值守下單。
3.  **即時監控:** 透過 LINE 與 Slack 推播進出場訊號與帳務狀況。
4.  **高擴充性:** 設計了易於維護的架構，可快速抽換或新增交易策略。

## 實際成效 (Results)
* **策略落地:** 2022 年底上線 2 項策略，2023-2024 年間新增 3 項，目前累計 5 項策略穩定運行。
* **投資回報:** **2023 與 2024 兩年的投資報酬率 (ROI) 皆達到 120%**。
